# Passione Mobile

Мобильное приложение для системы ресторанов Passione на Android (Kotlin).

## Требования

- Android Studio Hedgehog или новее
- JDK 8 или выше
- Android SDK 24+

## Открытие проекта

1. Откройте Android Studio
2. Выберите "Open an existing project"
3. Укажите папку `apps/mobile`

## Запуск

1. Подключите Android устройство или запустите эмулятор
2. Нажмите "Run" в Android Studio
3. Приложение установится и запустится на устройстве

## Сборка APK

```bash
./gradlew assembleDebug
```

APK будет находиться в `app/build/outputs/apk/debug/`
